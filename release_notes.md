## 📸 가족 추억 보관 상자 v21

### ✨ 주요 기능
- 🔒 **보안 100%** — 사진이 클라우드에 올라가지 않습니다. 집 Wi-Fi 안에서만 이동하므로 해킹 위험 제로
- 🖼️ **원본 무손실 전송** — 화질 압축 없이 원본 그대로 전송 (SHA-256 무결성 검증)
- 📁 **자동 폴더 정리** — 외장하드에 날짜_장소명 폴더 자동 생성 (예: `2024-03-15_하남스타필드`)
- 📱 **핸드폰 용량 절약** — 전송 후 원본 삭제 가능, 앱에서 99% 용량 줄인 미리보기로 확인 가능
- 🗺️ **GPS 위치 자동 변환** — 사진 속 GPS 좌표를 실제 장소 이름으로 자동 변환 (Nominatim/OpenStreetMap, 무료)
- 💾 **외장하드 자동 감지** — USB 외장하드 꽂으면 자동으로 사진 이관
- 🌐 **웹 전송 지원** — 앱 설치 없이 핸드폰 브라우저에서도 사진 전송 가능
- 🗑️ **백업 갤러리 선택/삭제** — 날짜별 전체 선택, 개별 체크, 서버에서 사진 삭제 가능 *(v21 신규)*
- 💰 **100% 무료** — 클라우드 비용, 월정액 없음

---

### 📱 앱 vs 🌐 웹 — 사진을 보내는 2가지 방법

사진을 보내는 방법이 **앱**과 **웹** 두 가지 있습니다.

- **앱**: 아래 APK를 핸드폰에 설치
- **웹**: 앱 설치 없이, 핸드폰 브라우저에서 `http://192.168.x.x:8000/web/` 접속 (서버 실행 시 화면에 주소가 표시됩니다)

| | 📱 앱 (APK 설치) | 🌐 웹 (브라우저) |
|--|---|---|
| **설치** | APK 설치 필요 | ❌ 설치 불필요 |
| **사진 전송** | ✅ | ✅ |
| **갤러리에서 바로 선택** | ✅ 편하게 선택 | ❌ 파일 탐색기에서 선택 |
| **전송된 사진 표시** | ✅ 초록 체크 표시 | ❌ 표시 없음 |
| **전송 후 원본 삭제** | ✅ 앱에서 바로 삭제 | ❌ 직접 삭제 |
| **썸네일 미리보기** | ✅ 백업 갤러리 | ❌ 없음 |
| **전송 진행률** | ✅ 실시간 | ✅ 실시간 |
| **추천 대상** | 본인 (주로 쓰는 사람) | 가족·친구 (가끔 보내는 사람) |

> 💡 본인은 **앱**으로 쓰고, 앱 설치가 귀찮은 가족에게는 **웹 주소만 알려주면** 됩니다!

> ⚠️ **웹 주소는 왜 링크가 아닌가요?**
> 
> `http://192.168.x.x:8000/web/` 에서 x.x 숫자는 **집마다 다릅니다.**
> 이건 인터넷 주소(naver.com)가 아니라, **집 Wi-Fi 안에서만 존재하는 내부 주소**예요.
> 서버를 실행하면 파란 창에 본인의 주소가 표시됩니다. 그 주소를 가족에게 알려주세요.
> 집 Wi-Fi가 바뀌지 않는 한 주소는 항상 같으니, 한 번만 알려주면 됩니다.

#### 📱 핸드폰 용량 절약 예시

사진을 서버로 전송한 뒤 핸드폰에서 원본을 삭제해도, **앱에서 아주 작은 미리보기(썸네일)**로 사진을 계속 볼 수 있습니다.

| 사진 수 | 원본 용량 | 앱에 남는 썸네일 용량 | 절약되는 용량 |
|--------|----------|-------------------|------------|
| 1,000장 | 약 5GB | 약 30MB | **4.97GB 절약** (99.4%) |
| 10,000장 | 약 50GB | 약 300MB | **49.7GB 절약** |
| 100,000장 | 약 500GB | 약 3GB | **497GB 절약** |

> 예를 들어 핸드폰에 사진 1만 장(50GB)이 있다면, 전송 후 원본을 지우면 **49.7GB가 비워집니다.**
> 앱에는 300MB짜리 작은 미리보기만 남아서, 언제든 사진을 훑어볼 수 있어요.

### 📥 설치 방법 (처음부터 끝까지)

이 앱은 **핸드폰 앱 + 노트북 서버** 두 가지를 모두 설치해야 동작합니다.

> 💡 **왜 서버도 설치해야 하나요?**
> 
> 이 앱은 구글 포토처럼 인터넷 서버를 쓰지 않고, **내 노트북이 서버 역할**을 합니다.
> 핸드폰에서 사진을 보내면 노트북이 받아서 정리해주는 구조라서,
> 사진을 받아줄 노트북 프로그램(서버)이 필요한 거예요.
> USB 케이블로 파일을 옮길 때 PC가 켜져 있어야 하는 것과 같은 원리입니다.

---

#### 🔹 1단계: 노트북에 Python 설치하기

Python은 서버를 실행하기 위한 프로그래밍 도구입니다. (무료)

1. 브라우저에서 **https://www.python.org/downloads/** 접속
2. 노란색 **"Download Python 3.xx.x"** 버튼 클릭하여 다운로드
3. 다운로드된 설치 파일 실행
4. ⚠️ **중요!** 설치 화면 맨 아래에 **"Add Python to PATH"** 체크박스가 있습니다. **반드시 체크**하세요!
5. **"Install Now"** 클릭 → 설치 완료까지 기다리기

**설치 확인 방법:**
- 키보드에서 `Windows키 + R` 누르기
- `cmd` 입력 후 엔터
- 검은 창에 `python --version` 입력 후 엔터
- `Python 3.xx.x`가 나오면 성공! ✅

---

#### 🔹 2단계: 서버 코드 다운로드하기

1. 이 GitHub 페이지 상단으로 이동 (https://github.com/abraxass0511-lab/family-photo-archive)
2. 초록색 **"<> Code"** 버튼 클릭
3. **"Download ZIP"** 클릭
4. 다운로드된 ZIP 파일을 **바탕화면에 압축 풀기** (우클릭 → 여기에 풀기)
5. `family-photo-archive-master` 폴더가 생깁니다

---

#### 🔹 3단계: 서버 준비하기 (최초 1회만 하면 됩니다)

1. 위에서 풀은 폴더 안의 **`server`** 폴더를 열기
2. 폴더 빈 공간에서 **Shift + 마우스 우클릭** → **"여기서 PowerShell 창 열기"** 클릭
3. 파란 창이 열리면, 아래 명령어를 **한 줄씩** 복사해서 붙여넣고 엔터:

**첫 번째 줄** (가상환경 만들기):
```
python -m venv venv
```

**두 번째 줄** (가상환경 켜기):
```
.\venv\Scripts\activate
```

**세 번째 줄** (필요한 프로그램 설치):
```
pip install -r requirements.txt
```

> ⏳ 세 번째 명령어는 필요한 프로그램들을 자동으로 다운로드합니다.
> 인터넷 속도에 따라 **3~10분** 걸릴 수 있어요.
> 화면에 글자가 계속 나오고 있으면 정상입니다. 기다리면 됩니다!

---

#### 🔹 4단계: 서버 실행하기

1. `family-photo-archive-master` 폴더에 있는 **`start_server.ps1`** 파일을 **우클릭**
2. **"PowerShell에서 실행"** 클릭
3. 파란 창이 열리면서 아래와 비슷한 내용이 표시됩니다:
```
  Phone Access URL:
  http://192.168.0.5:8000/web/
```
4. 여기 나오는 **`http://192.168.0.5:8000`** ← 이 주소를 메모하세요!
   (숫자는 사용자마다 다릅니다)

> 💡 이 파란 창을 닫으면 서버가 꺼집니다.
> **사진을 전송하는 동안에는 이 창을 열어두세요.** 전송이 끝나면 닫아도 됩니다.

---

#### 🔹 5단계: 핸드폰에 앱 설치하기

1. 핸드폰 브라우저에서 이 페이지에 접속합니다
2. 아래쪽 **Assets** 섹션에서 **`포토백업_v21.apk`** 를 터치하여 다운로드
3. 다운로드 완료 후 알림을 터치해서 설치 시작
4. **"출처를 알 수 없는 앱"** 경고가 뜨면:
   - "설정" 터치 → "이 출처 허용" 켜기 → 뒤로 → 다시 설치
5. 설치 완료 → **"열기"** 터치

---

#### 🔹 6단계: 앱에서 서버 연결하기

1. 앱을 열면 **설정** 탭이 보입니다
2. **서버 주소** 칸에 4단계에서 메모한 주소를 입력합니다
   - 예시: `http://192.168.0.5:8000`
3. **"연결 테스트"** 버튼을 눌러서 ✅ 연결 성공이 나오는지 확인
4. 연결 성공이면 **전송 탭**으로 이동 → 갤러리에서 사진 선택 → 전송!

> ⚠️ **핸드폰과 노트북이 같은 Wi-Fi에 연결되어 있어야 합니다!**
> 집 와이파이에 둘 다 연결하세요.

### ❓ 자주 묻는 질문

**Q. 노트북 꺼져 있어도 전송 돼?**
→ 아닙니다. 노트북이 켜져 있고 서버(파란 창)가 실행 중일 때만 전송 가능합니다.

**Q. 사진 화질이 떨어져?**
→ 아닙니다. 원본 그대로 1바이트도 변하지 않고 전송됩니다.

**Q. 해킹 위험은?**
→ 사진이 인터넷(클라우드)을 거치지 않고 집 Wi-Fi 안에서만 이동하므로 보안 100%입니다.

**Q. 핸드폰에서 원본 지우면 다시 못 봐?**
→ 앱에서 99% 용량 줄인 썸네일(미리보기)로 계속 볼 수 있어요. 원본은 외장하드에 있습니다.

**Q. 카페 Wi-Fi에서도 돼?**
→ 핸드폰과 노트북이 **같은 Wi-Fi**에 연결되어 있어야 합니다.
