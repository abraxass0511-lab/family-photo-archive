# 📸 가족 추억 보관 상자

> **클라우드 없이, 내 외장하드에 원본 그대로 보관하는 가족 사진 관리 앱**

핸드폰 사진을 Wi-Fi로 노트북에 보내면, **날짜와 촬영 장소 이름으로 폴더를 자동 생성**해서 외장하드에 정리해주는 앱입니다.
구글 포토처럼 클라우드를 쓰지 않아서 **월정액도 없고, 사진이 인터넷 밖으로 나가지 않아 보안이 100%**입니다.

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| 🔒 **보안 100%** | 사진이 인터넷에 올라가지 않음. 집 Wi-Fi 안에서만 이동 |
| 📱 **폰 용량 해방** | 전송 후 원본 삭제, 99% 용량 절약된 썸네일로 미리보기 |
| 🖼️ **원본 무손실** | 화질 압축 없이 원본 그대로 전송 (SHA-256 무결성 검증) |
| 📁 **자동 폴더 정리** | `2024-03-15_하남스타필드` 형식으로 날짜+장소별 자동 정리 |
| 🗺️ **GPS → 장소명 변환** | 사진 속 GPS 좌표를 실제 장소 이름으로 자동 변환 (Nominatim, 무료) |
| 👨‍👩‍👧‍👦 **AI 얼굴 인식** | 사진 속 인물을 자동으로 태그 |
| 💾 **외장하드 자동 감지** | USB 꽂으면 자동으로 사진 이관 시작 |
| 💰 **100% 무료** | 클라우드 비용, 월정액 없음 |

## 🏗️ 구조

```
📱 핸드폰 앱 (Flutter)
   │
   │  Wi-Fi 전송 (같은 Wi-Fi)
   ▼
💻 노트북 서버 (Python FastAPI)
   │
   │  USB 외장하드 꽂으면 자동 이관
   ▼
💾 외장하드
   ├── 2024-03-15_하남스타필드/
   ├── 2024-05-01_제주도_서귀포/
   └── 2024-12-25_집/
```

# 📥 설치 가이드 (처음부터 끝까지)

> ⚠️ 이 앱은 **핸드폰 앱 + 노트북 서버** 두 가지를 모두 설치해야 동작합니다.

## 1단계: 노트북에 Python 설치하기

Python은 서버를 실행하기 위한 프로그래밍 도구입니다.

1. 브라우저에서 **https://www.python.org/downloads/** 에 접속
2. 노란색 **"Download Python 3.xx.x"** 버튼 클릭
3. 다운로드된 설치 파일 실행
4. ⚠️ **중요!** 설치 화면 맨 아래에 **"Add Python to PATH"** 체크박스가 있습니다. **반드시 체크**하세요!
5. "Install Now" 클릭 → 설치 완료까지 기다리기
6. 설치 완료 화면에서 "Close" 클릭

### 설치 확인 방법
1. 키보드에서 `Windows키 + R` 누르기
2. `cmd` 입력 후 엔터
3. 검은 창에 `python --version` 입력 후 엔터
4. `Python 3.xx.x`가 나오면 성공! ✅

---

## 2단계: 서버 코드 다운로드하기

### 방법 A: 다운로드 (쉬운 방법)
1. **https://github.com/abraxass0511-lab/family-photo-archive** 접속
2. 초록색 **"<> Code"** 버튼 클릭
3. **"Download ZIP"** 클릭
4. 다운로드된 ZIP 파일을 바탕화면에 풀기 (우클릭 → 여기에 풀기)
5. `family-photo-archive` 폴더가 생깁니다

### 방법 B: Git 사용 (개발자용)
```
git clone https://github.com/abraxass0511-lab/family-photo-archive.git
```

---

## 3단계: 서버 준비하기

1. `family-photo-archive` 폴더 안의 `server` 폴더를 열기
2. 폴더 빈 공간에서 **Shift + 마우스 우클릭** → **"여기서 PowerShell 창 열기"** 클릭
3. 아래 명령어를 **한 줄씩** 복사해서 붙여넣고 엔터:

```
python -m venv venv
```
```
.\venv\Scripts\activate
```
```
pip install -r requirements.txt
```

> ⏳ 마지막 명령어는 필요한 프로그램들을 자동으로 다운로드하는 과정입니다. 인터넷 속도에 따라 3~10분 걸릴 수 있어요. 기다리면 됩니다!

---

## 4단계: 서버 실행하기

### 실행 방법
1. `family-photo-archive` 폴더에 있는 `start_server.ps1` 파일을 **우클릭**
2. **"PowerShell에서 실행"** 클릭
3. 파란 창이 열리면서 아래와 비슷한 내용이 표시됩니다:

```
  ========================================
   Photo & Video Transfer Server
  ========================================

  Phone Access URL:
  http://192.168.0.5:8000/web/

  Close this window to stop the server.
```

4. **`http://192.168.0.5:8000`** ← 이 주소가 핸드폰 앱에서 입력할 서버 주소입니다!
   (숫자는 사용자마다 다릅니다)

> 💡 이 파란 창을 닫으면 서버가 꺼집니다. **사진을 전송하는 동안에는 이 창을 열어두세요.**

---

## 5단계: 핸드폰에 앱 설치하기

1. 핸드폰 브라우저에서 아래 링크 접속:
   **https://github.com/abraxass0511-lab/family-photo-archive/releases**
2. 가장 위에 있는 최신 버전의 **`포토백업_v20.apk`** 를 터치해서 다운로드
3. 다운로드 완료 후 알림을 터치해서 설치 시작
4. **"출처를 알 수 없는 앱"** 경고가 뜨면:
   - "설정" 터치
   - "이 출처 허용" 또는 "알 수 없는 앱 설치" 켜기
   - 뒤로 가서 다시 설치
5. 설치 완료 → **"열기"** 터치

---

## 6단계: 앱에서 서버 연결하기

1. 앱을 열면 **설정** 화면이 보입니다
2. **서버 주소** 칸에 4단계에서 나온 주소를 입력합니다
   - 예시: `http://192.168.0.5:8000`
3. **"연결 테스트"** 버튼을 눌러서 ✅ 연결 성공이 나오는지 확인
4. 연결 성공이면 **전송 탭**으로 이동 → 갤러리에서 사진 선택 → 전송!

> ⚠️ **핸드폰과 노트북이 같은 Wi-Fi에 연결되어 있어야 합니다!**

---

## 7단계: 외장하드 자동 정리 (선택)

1. 사진이 노트북 바탕화면의 `사진모음집` 폴더에 쌓입니다
2. 외장하드를 USB에 꽂으면 → **자동으로 외장하드에 정리됩니다**
3. 폴더 형식: `2024-03-15_하남스타필드` (날짜_장소명)

> 외장하드 경로는 `server` 폴더의 `.env` 파일에서 변경할 수 있습니다.

## ❓ 자주 묻는 질문

### 노트북 꺼져 있어도 전송 돼?
> **아니요.** 노트북이 켜져 있고, 서버 프로그램(파란 창)이 실행 중일 때만 전송할 수 있어요.

### 사진 화질이 줄어들어?
> **아니요.** 원본 그대로 1바이트도 변하지 않고 전송됩니다.

### 핸드폰에서 원본 지우면 다시 못 봐?
> 앱에서 **용량 99% 줄인 미리보기(썸네일)**로 계속 볼 수 있어요. 원본이 필요하면 외장하드에서 꺼내면 됩니다.

### 해킹 위험은?
> 사진이 **인터넷을 거치지 않고** 집 Wi-Fi 안에서만 이동하기 때문에 해킹 위험이 없습니다. 보안 100%.

### 카페 Wi-Fi에서도 되나?
> 핸드폰과 노트북이 **같은 Wi-Fi**에 연결되어 있어야 합니다.
